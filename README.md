# Jarvis Mobile

Ein experimenteller Android-Assistent für das Xiaomi 14T Pro.

## Erstes Ziel

Ein lokales **Aktionszentrum**, das Alltagssituationen mit einem Tastendruck startet:

- Wohnmobilmodus
- Fokusmodus
- PC-Steuerung
- Dokumentensuche

Der erste Prototyp enthält eine native Android-App mit Jetpack Compose. Er startet vorhandene Navigations-Apps über Android-Intents und bildet die Grundlage für spätere Automatisierungen.

## Technik

- Kotlin
- Jetpack Compose
- Material 3
- Android 16 kompatibel
- Keine Cloud-Abhängigkeit im ersten Schritt

## Nächste Ausbaustufen

1. Bluetooth-Erkennung des Fiat Ducato
2. Automatischer Wohnmobilmodus
3. Lokale Erinnerungen und Fokus-Timer
4. PC-Verbindung im lokalen Netzwerk
5. Lokale KI für Dokumente und Fotos
